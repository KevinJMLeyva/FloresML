# FloresML
Clasificador de flores es mediante la utilización de machine learning poder clasificar fotografías de flores y determinar si la fotografía es alguna de las siguientes flores “Black eyed Susan, Calendula, California poppy, Common Daisy, Coreopsis, Daffodil, Dandelion, Iris, Magnolia, Rose, Sunflower, Tulip, Water Lily”.
